# 1. Project Vision

Confirmed Decision

Build an Excalidraw-like infinite whiteboard specifically for modeling money flow.

Users create visual "money boxes" connected by animated proportional ribbons representing cash movement.

Each node represents an account, category, income source, savings goal, investment, debt payment, transfer, expense, or any other financial entity.

Connections represent money flowing between nodes.

The application continuously calculates monthly equivalents regardless of original frequency.

The whiteboard becomes a living financial model.

Ultimate goal:

Allow users to experiment with financial scenarios visually rather than through spreadsheets.

# 23. Final Summary

FlowBudget is an offline-first visual financial modeling application inspired by Excalidraw and Sankey diagrams. Users construct directed graphs representing the flow of money between financial entities. Nodes represent accounts or categories, while proportional ribbons represent cash flow. The application continuously recalculates monthly values, supports formulas referencing other nodes, and enables rapid "what-if" scenario analysis without requiring bank integrations, accounts, or a backend.

The first implementation milestone is a polished local-first React application featuring an infinite whiteboard, draggable nodes, ribbon connections, frequency normalization, JSON import/export, and local persistence. Once that foundation is complete, the signature "Flow Mode" animation should be added to visually show money traveling through the graph, reinforcing the application's identity as a financial modeling tool rather than a traditional budgeting application.
